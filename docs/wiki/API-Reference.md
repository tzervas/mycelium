# API Reference

Mycelium ships **two** complementary API references for the Rust workspace:

1. **rustdoc** — the full generated HTML API docs for every crate.
2. **the agent API index** — a committed, deterministic, grep-friendly symbol table for fast lookups
   (used heavily by AI agents working in the repo).

## rustdoc (generated HTML)

Build it locally:

```sh
just docs        # cargo doc for the workspace (output under target/doc/, NOT committed)
# or directly:
cargo doc --no-deps --workspace --open
```

Each crate's entry point is `target/doc/<crate_underscored>/index.html` (e.g.
`target/doc/mycelium_core/index.html`). The doc comments are the authoritative per-item reference —
they carry the per-op guarantee tags and the design-doc cross-references.

**Published copy.** The repository's `publish-docs` GitHub Action (manual dispatch) builds this
rustdoc and deploys it to GitHub Pages; once enabled it is browsable at the repository's Pages URL.
(Mycelium keeps CI manual-dispatch-only by policy, so the docs are published on demand, not on every
push.)

## The agent API index (committed)

`docs/api-index/` holds two committed artifacts generated by `tools/docgen/code_index.py`:

- [`INDEX.md`](https://github.com/tzervas/mycelium/blob/main/docs/api-index/INDEX.md) — a
  grep-friendly table grouped by crate (symbol · file:line · summary · guarantee tag).
- `index.json` — the machine-readable symbol table.

Regenerate after any public-API change:

```sh
just docs-index   # python3 tools/docgen/code_index.py
```

**Transparency note.** The index is an `Empirical`/`Declared` line/regex heuristic — **source is
ground truth**. Re-exports, macro-generated, and cfg-gated items appear in a `flagged` section (G2:
never silently dropped). Use the index to find *where to read*, not as an authoritative reference.

## Committed API baselines

`docs/spec/api/<crate>.txt` are the committed public-API snapshots the `api` gate checks against
(`cargo public-api`). They are the contract for what each crate exposes (KC-3); a diff there is a
reviewed part of a change.

## Per-crate orientation

Every crate also has a short `README.md` (see the [Crate Index](Crate-Index)) for a 30-second
orientation before diving into rustdoc.
