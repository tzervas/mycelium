## 4. Stress questions for contamination (maintainer + implementers)

Use these as acceptance probes for any isolation slice:

1. **Single Declared leaf:** Does one `Declared` import strip an entire Exact pipeline, or stop at
   a meet-boundary / quarantine with EXPLAIN?
2. **Failed swap check:** Can a `NotValidated` or `Err` path type or display as Exact success in
   certified mode? (Must be **no**.)
3. **Laundry seal:** Is there any surface that remints stronger without predicate, cert, or basis?
   (Must be **no**.)
4. **Quality kill:** Does "safe" configuration force all values Declared and disable Exact cores?
   (Must be **no** — separate paths + partitions.)
5. **Dataset export:** Can a mixed-grade table ship as a single silent floor? (Must be **no** —
   partitions or explicit weak export.)
6. **Dynamic boundary:** If isolation decision is runtime, is the EXPLAIN package always generated?
   (Must be **yes**.)
7. **Mode firewall:** Can `fast`-floored claims meet into a `certified` Exact demand without seal?
   (Must be **no**.)
8. **Tooling greenwash:** Do transpile/LSP actions invent Proven/Exact to clear a metric? (Must be
   **no** — candidates only; tags stay Declared until witnessed.)
