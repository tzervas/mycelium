## Scope

Grow the skeletal v0 runtime (`crates/mycelium-std-runtime`, ADR-020 R1 API) into a real,
stable concurrency runtime — required for a fully-usable language at 1.0.0. The R1 slice
(`Colony`/`Scope`/`Task`/`Network`) is landed (M-521); E7-2's `hypha`/`colony`/`fuse`/`reclaim`
L1 constructs activate the vocabulary. This epic makes them *execute* end-to-end: a real
scheduler on OS threads, the full RFC-0008 vocabulary running, deadlock-freedom enforced, the
reclamation model (RFC-0027) live, and structured-concurrency supervision with never-silent
failure propagation.

**Epic issues:** E12-1 (parent epic), M-709, M-710, M-711, M-712, M-713.

## Swarm / parallelization pattern

**Sonnet orchestrator + parallel leaf agents (one per issue), octopus-merge.**

Ownership split:
- **Orchestrator** owns (within this wave): workspace `Cargo.toml`, its **design docs**
  (`docs/rfcs/RFC-0008-*`, `docs/rfcs/RFC-0027-*.md` once it advances, `docs/spec/stdlib/runtime.md`)
  and its **per-crate API baseline** (`docs/spec/api/mycelium-std-runtime.txt`). The **cross-track
  release-surface files** (`CHANGELOG.md`, `docs/adr/ADR-022`, `docs/Doc-Index.md`,
  `tools/github/issues.yaml`, `docs/api-index/`) are **not edited on this head** — they follow the
  `r10`↔`rel10` deconfliction (kickoffs README §Cross-track deconfliction): r10 **FLAGs** its
  CHANGELOG entry + its "T3 → gate-met" status up to `rel10` / the integrator.
- **Leaf M-709** owns: `crates/mycelium-std-runtime/` scheduler (OS-thread pool, fairness,
  backpressure).
- **Leaf M-710** owns: runtime vocabulary execution — `fuse`/`reclaim`/`tier`/`mesh`/`xloc`/
  `cyst` elaboration paths in `crates/mycelium-l1/src/elab.rs` + runtime dispatch (coordinate
  with E7-2's leaf for the L1 surface; read-only on E7-2's files).
- **Leaf M-711** owns: deadlock-freedom check/enforcement in
  `crates/mycelium-std-runtime/src/` — the RFC-0008 sweep-order property test + the
  never-silent `Deadlock` error path.
- **Leaf M-712** owns: `crates/mycelium-std-runtime/src/` reclamation model (post RFC-0027
  Accepted); the EXPLAIN record for reclamation events.
- **Leaf M-713** owns: `crates/mycelium-std-runtime/src/` supervision + cancellation (structured
  concurrency tree, `CancelToken` propagation, never-silent failure paths).

**Cross-track deconfliction (with `rel10`, the docs/release track):** directory-disjoint (code vs
prose) — develop in parallel. r10 **FLAGs** its `CHANGELOG.md` entry and its "T3 → gate-met" status
**up to `rel10`** rather than editing `CHANGELOG.md` / `docs/adr/ADR-022` directly; `rel10` reconciles
the ADR-022 §5 gate cell and integrates the release **after** r10's T3 lands. r10 owns its own
`docs/spec/api/mycelium-std-runtime.txt` baseline; `docs/api-index/` is regenerated by the integrator
on the merged state. Full table: kickoffs README §Cross-track deconfliction.

Pre-flight: align head with `main` (`git fetch origin; git merge --ff-only origin/main`);
prune stale worktrees; push the head branch before spawning any leaf.

Swarm failure mitigations (per CLAUDE.md):

## Definition of Done

- [ ] A real OS-thread scheduler runs `Colony`/`Scope` tasks with documented fairness and
  backpressure properties, tagged `Empirical` (in-repo property test; no `Proven` without a
  mechanized proof).
- [ ] The RFC-0008 R1 vocabulary executes end-to-end: `fuse`/`reclaim`/`tier` elaborate and
  dispatch; no `Residual` for any R1 construct (RT1–RT7 invariants hold by test).
- [ ] Deadlock is never silent: the RFC-0008 §4.3 sweep-order property is a checked assertion;
  a violation is an explicit `Deadlock` error (G2), not a hang.
- [ ] The RFC-0027 reclamation model is live: EXPLAIN records for reclamation events; no silent
  drops; the reclamation cascade respects sweep order.
- [ ] Structured-concurrency supervision is end-to-end: a cancelled `Colony` propagates failure
  to every child; the `CancelToken` path is exercised by a never-silent property test.
- [ ] `just check` green across all modified crates; guarantee tags honest per VR-5; no
  `Proven` tag without a mechanized proof.
- [ ] Orchestrator-owned files reconciled: `CHANGELOG.md` entry, `issues.yaml` status updated
  (M-709…M-713 → done), RFC-0027 status note added, `docs/api-index/` regenerated.
